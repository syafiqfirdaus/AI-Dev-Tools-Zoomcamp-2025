# Collaborative Coding Platform

A real-time collaborative coding interview platform built with React, Express, and Socket.io.

## Features

- Real-time code collaboration
- Syntax highlighting for JavaScript and Python
- Code execution (Coming soon)

## Getting Started

### Prerequisites

- Node.js installed

### Installation

1. Install dependencies:

   ```bash
   npm install
   cd client && npm install
   cd ../server && npm install
   ```

### Running the Application

Run both client and server:

```bash
npm run dev
```

The client will be available at `http://localhost:5173` and the server at `http://localhost:3001`.

### Running Tests

Run integration tests:

```bash
cd server
npm test
```
